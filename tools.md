# Tools & Packages

Before we can begin patching ROBLOX, we need to download and set up some essential tools & system packages. This section will provide you installation instructions for everything you need to get started, for all versions.

## Required Tools & Packages
- [Visual Studio Redistributables 2015–2022](https://aka.ms/vs/17/release/vc_redist.x64.exe)  
  Install both **x64** and **x86** versions. This is required by RCCService and some of these tools to function.
- [HxD Hex Editor](https://mh-nexus.de/en/downloads.php?product=HxD) (Hex Editor)  
  HxD is a hex editor that is used in many of the patches on this wiki. We recommend it for simplicity, however you can use any hex editor you prefer.
- [x32dbg/x64dbg](https://x64dbg.com/) (x86/x64 Debugger)
- [IDA Free](https://www.hex-rays.com/products/ida/support/download_freeware/) (Disassembler)
- [Certificate Generator](https://github.com/worships/rbxsign)
- [Rust](https://www.rust-lang.org/tools/install) (Latest Stable Version)
- [Bun](https://bun.sh/) (Latest Version)
- [NodeJS](https://nodejs.org/en/download/) (If you want to use npm/nodejs over bun)

## Optional Tools & Packages
- [Visual Studio Code](https://code.visualstudio.com/) (Code Editor)
- [Git](https://git-scm.com/downloads) (Version Control)
- [Roblox Auto Patcher](https://github.com/worships/roblox-auto-patcher) (Auto Patching Tool)
- [Lure](https://github.com/lrre-foss/lure) (DLL Client/Server Patch)
